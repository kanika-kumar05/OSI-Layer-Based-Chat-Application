# Multi-Layer Packet Filtering System

A Python-based simulation of a 7-layer OSI model firewall with advanced algorithmic filtering.

## 🚀 Quick Start
Run the simulation:
```bash
python main.py
```

## 🛡️ Features
- **OSI Simulation**: Full L1-L7 stack.
- **Algorithms**: Bloom Filter (L3), Trie & Rabin-Karp (L7).
- **Security**: Content, Port, and IP-based filtering.
- **Visuals**: Color-coded console output.

## 🛠️ Requirements
- Python 3.10+
- No external libraries needed!
